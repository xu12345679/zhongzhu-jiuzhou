# 主角 · 苏北辰

> *"上一世我守到国破人亡，这一世——我要让这片土地，再无裂痕。"*

## 基本信息

- **姓名：** 苏北辰
- **前世身份：** 大夏王朝镇北军元帅，"北境之壁"
- **今世身份：** 苍玄界·大虞王朝·江州苏家嫡长子（家族没落）
- **外貌：** 十七岁，身形修长，面容清秀但不失棱角，一双眼睛格外沉静，偶尔流露不属于这个年龄的沧桑
- **性格：** 沉稳内敛、杀伐果断、重情重义、心怀天下

## 前世（记忆残片）

他是大夏王朝最后的名将。十七岁从军，二十五岁封帅，率镇北军抵挡蛮族入侵十余年。然而朝堂腐败，粮草不继，他被自己守护的人背叛——朝中权贵为求和，断了他的补给。最后一战，他以三千残兵对阵五万蛮骑，战至力竭，坠马而亡。

**临终遗憾：** 不是战死，而是死于自己人的背叛。这让他重生后对人心的判断格外敏锐，也格外痛恨内耗与短视。

## 今世

大虞王朝江州苏氏嫡长子。苏家五年前因卷入"天关军械案"被削爵夺田，家道中落。父亲苏远山郁愤成疾，卧床三年。母亲早逝。还有一个妹妹苏晚晴（14岁，聪慧懂事）。

苏北辰幼年测试天赋评为"中下等"，被族中旁支欺辱，性格原本怯懦。但真正的苏北辰在三年前落水高烧后就已"死了"——现在的他，是那个北境军神的魂魄。

## 性格细描

| 维度 | 表现 |
|------|------|
| **外表** | 冷静从容，喜怒不形于色。但真正动怒时，眼神冷得像寒冬 |
| **内在** | 强烈的使命感，对战友和部下极为护短 |
| **说话风格** | 简洁有力，不废话。偶尔毒舌，精准到位 |
| **弱点** | 难以信任权贵；见到无谓牺牲会触发前世创伤 |
| **战斗风格** | 战术优先，从不恋战。能用三分力绝不花五分，该拼时绝不犹豫 |
| **感情观** | 前世一生未娶，心中只有家国。这一世遇到那个对的人时，自己都没准备好 |

## 修炼天赋

- **体质：** 前世神魂强大的残余让他对天地法则有天然亲和力
- **破境速度：** 远超常人，但每次突破都会触发前世记忆碎片
- **独特能力：** 战术直觉——可以将战场判断力转化为战斗嗅觉，预判对手招式
- **核心功法：** 后期自创融合前世兵法与武道感悟的独门功法
