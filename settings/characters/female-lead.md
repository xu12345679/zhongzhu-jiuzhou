# 女主 · 林清雪

> *"我林家世代镇守天关，我不需要谁来保护——我要的是能并肩作战的人。"*

## 基本信息

- **姓名：** 林清雪
- **身份：** 大虞王朝·镇南侯林啸天之女，天关军校优秀学员
- **外貌：** 双十年华（20岁），身姿挺拔如松，眉目清冷，有种军旅子弟特有的英气。素面朝天却自有一股清冽之美
- **性格：** 外冷内热、坚韧独立、是非分明、坦荡磊落

## 背景

镇南侯世代镇守天关南段，林家是军中世家。林清雪是林啸天的独女，自幼随父习武，十二岁便能拉开三石弓。她不依靠家世，以真才实学考入天关军校——培养边防军官的最高学府。

母亲在她幼年时死于一次虚魇突袭，这让她对虚魇有着刻骨的仇恨，也让她比同龄人更成熟。

## 性格细描

| 维度 | 表现 |
|------|------|
| **外表** | 清冷自持，严肃认真，军校里人称"冰山校花" |
| **内在** | 重情重义，对认可的人会毫无保留地维护 |
| **说话风格** | 干脆利落，不拖泥带水。欣赏同样干脆的人 |
| **行动力** | 极强，想到就做，从不犹豫 |
| **弱点** | 对父亲有执念——想证明自己不比男儿差；偶尔过于刚直不够圆滑 |
| **感情观** | 从不觉得自己需要被保护，但也不排斥真心。她要的是平等的、并肩的伴侣 |

## 与主角的关系发展

- **初识：** 天关军校入学考核中相遇。她对主角一开始是怀疑和审视
- **转折：** 共同执行任务中发现主角的战术眼光和对天下的担当，态度转变
- **深入：** 经历生死后彼此信任，从战友到知己，从知己到生死与共
- **定位：** 不是需要被保护的花瓶——她是能在战场上与主角背靠背的战友

## 修炼境界

- 出场：凝元境中期（20岁凝元，已属天才）
- 随剧情成长，始终与主角保持合理差距
